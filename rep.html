System File Checker

Der System File Checker (SFC) ist ein integriertes Windows-Tool, das dazu dient, beschädigte oder fehlende Systemdateien zu erkennen und zu reparieren. Es überprüft die Integrität der geschützten Systemdateien und ersetzt beschädigte Dateien durch eine zwischengespeicherte Kopie, die sich in einem komprimierten Ordner auf der Festplatte befindet.

sfc /scannow

DISM (Deployment Image Servicing and Management) Repair Command

DISM ist ein Befehl, der verwendet wird, um das Windows-Image zu reparieren. Er kann verwendet werden, um beschädigte Systemdateien zu ersetzen und das Betriebssystem zu reparieren.

DISM /Online /Cleanup-Image /RestoreHealth

CHKDSK (Check Disk) Command

CHKDSK ist ein Befehl, der verwendet wird, um die Festplatte auf Fehler zu überprüfen und diese zu reparieren.

CHKDSK C: /f /r

Network Reset Command

Der Befehl "netsh winsock reset" dient dazu, den Winsock-Katalog zurückzusetzen und Netzwerkprobleme zu beheben.

netsh winsock reset

Flush DNS Cache

Der Befehl "ipconfig /flushdns" dient dazu, den DNS-Cache zu leeren und Netzwerkprobleme zu beheben.

ipconfig /flushdns

Release IP Address

Der Befehl "ipconfig /release" dient dazu, die aktuelle IP-Adresse freizugeben.

ipconfig /release

Renew IP Address

Der Befehl "ipconfig /renew" dient dazu, eine neue IP-Adresse zu erhalten.

ipconfig /renew

Boot Repair

Der Befehl "bootrec /fixmbr" dient dazu, den Master Boot Record (MBR) zu reparieren. Der Befehl "bootrec /scannow" dient dazu, das System auf Fehler zu überprüfen. Der Befehl "bootrec /rebuildbcd" dient dazu, den Bootdatenbank-Eintrag zu rebuilden.

bootrec /fixmbr
bootrec /scannow
bootrec /rebuildbcd

Driver Verification

Der Befehl "verifier" dient dazu, die Treiber zu überprüfen und Probleme zu diagnostizieren.

verifier

Memory Diagnostics

Der Befehl "mdsched.exe" dient dazu, den Arbeitsspeicher auf Fehler zu überprüfen.

mdsched.exe

Power Efficiency Report

Der Befehl "powercfg /energy" dient dazu, einen Bericht über die Energieeffizienz des Systems zu erstellen.

powercfg /energy

Battery Health Report

Der Befehl "powercfg /batteryreport" dient dazu, einen Bericht über die Batteriegesundheit des Systems zu erstellen.

powercfg /batteryreport

Remove Temporary Files

Der Befehl "cleanmgr" dient dazu, temporäre Dateien zu entfernen und Speicherplatz freizugeben.

cleanmgr

Event Viewer

Der Befehl "eventvwr.msc" dient dazu, den Ereignisanzeiger zu öffnen und Ereignisse zu überprüfen.

eventvwr.msc

System Information

Der Befehl "msinfo32" dient dazu, Informationen über das System zu erhalten.

msinfo32
